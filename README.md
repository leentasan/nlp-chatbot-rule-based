## Setup & run untuk CLI dan integrasi
git clone https://github.com/leentasan/nlp-chatbot-rule-based.git <br>
cd nlp-chatbot-rule-based <br>
npm install <br>
npm start (CLI) <br>
node src/whatsapp.js (Integrasi Whatsapp) <br>
